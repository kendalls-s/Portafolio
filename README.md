## Descripción

Sitio web estático de tres páginas que funciona como portafolio personal del estudiante. Cumple con las tres historias de usuario definidas en la Tarea Corta 1.

---

## Estructura de archivos

```
portafolio/
├── index.html       # Página principal – información del estudiante (HU 1)
├── galeria.html     # Galería de proyectos (HU 2)
├── contacto.html    # Formas de contacto (HU 3)
└── styles.css       # Hoja de estilos compartida
```

---

## Páginas

### index.html — Información del estudiante
- Nombre y apellidos
- Fecha de nacimiento
- Avatar con iniciales
- Residencia
- Encabezado con menú de navegación
- Pie de página con enlaces a redes sociales

### galeria.html — Galería de proyectos
- Tarjetas con título, curso y descripción de cada proyecto
- Conserva encabezado y pie de página

### contacto.html — Formas de contacto
- Teléfono, correos electrónicos y LinkedIn
- Conserva encabezado y pie de página

---

## Cómo usar

1. Descargar los 4 archivos en una misma carpeta.
2. Abrir `index.html` en cualquier navegador.
3. Navegar entre páginas usando el menú del encabezado.

No requiere servidor ni instalación adicional.
